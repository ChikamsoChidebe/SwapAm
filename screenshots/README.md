## Instructions for Taking Screenshots

1. **Use Full HD Resolution (1920x1080)**
2. **Clear Browser Cache** before taking screenshots
3. **Use Demo Data** for consistent screenshots
4. **Capture Different States**:
   - Loading states
   - Success states
   - Error states
   - Empty states

## Screenshot Naming Convention
- Use kebab-case for file names
- Include page/feature name
- Add state if applicable (e.g., `login-loading.png`)
- Use PNG format for best quality
